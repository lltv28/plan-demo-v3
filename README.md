# plan-demo-v3

Mobile-first interactive demo of the Lucas AI plan flow — a single-file HTML site that walks through a scripted quiz, a generated plan reveal (with narrated video + captions), a paywall with auto-filling card sheet, and a Calendly-style booking flow.

## View it

Live on GitHub Pages: https://lltv28.github.io/plan-demo-v3/

## Run locally

```bash
python -m http.server 8032
```

Then open `http://127.0.0.1:8032/`.
